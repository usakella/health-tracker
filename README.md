# health-tracker


## About
A private, non-commercial family health tracking application. 
This tool sends daily SMS health check-in messages to immediate 
family members to help track symptoms and medical conditions over time.

## Message Recipients
- Immediate family members only (2 recipients)
- All recipients have provided explicit verbal and written consent
- No third parties, no marketing, no commercial use

## Opt-in Process
Recipients are the developer's parents who have been informed about 
and explicitly agreed to receive daily health check-in SMS messages. 
Consent was obtained via direct conversation prior to any messages 
being sent.

## Message Content
- Daily symptom check-in questions (pain levels, diet, exercise)
- Follow-up questions based on previous responses
- Maximum 5-10 messages per day per recipient
- No marketing or promotional content ever

## Privacy
All health data is stored privately and accessible only to 
immediate family members.
