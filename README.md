# Personal Health Tracker

A personal friends/family health tracking tool. Daily SMS check-ins are sent 
to either friends or family members who have explicitly opted in and consented to receive 
messages. This is a private, non-commercial project with no marketing 
or third-party messaging involved.

Recipients: friends/family members only
Message type: daily health symptom check-ins
Opt-in method: verbal and written consent from recipients
